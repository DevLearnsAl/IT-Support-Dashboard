# IT Support Dashboard

An interactive Power BI dashboard for analyzing IT support ticket data — tracking volume, resolution times, agent performance, and service trends to support data-driven decisions in an IT service desk.

## 📊 Overview

This dashboard provides a clear view of IT support operations, helping teams monitor workload, identify bottlenecks, and measure service quality. It answers questions such as:

- How many tickets are we receiving over time?
- What is the average resolution time?
- Which categories or issues are most common?
- How is each agent or team performing?

## 🔑 Key Metrics & Features

- **Total Tickets** — overall volume of support requests
- **Average Resolution Time** — how long tickets take to close
- **Tickets by Category / Priority** — breakdown of issue types
- **Tickets Over Time** — trend analysis by day, month, or quarter
- **Agent / Team Performance** — workload and resolution comparison
- **Interactive Filters** — slice by date, status, priority, and more
