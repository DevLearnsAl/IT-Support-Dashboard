# IT Support Dashboard

An interactive Power BI dashboard for analyzing IT support ticket data — tracking ticket volume, resolution times, agent performance, and service trends to support data-driven decisions in an IT service desk.

*Un dashboard interactivo de Power BI para analizar datos de tickets de soporte de IT — seguimiento del volumen de tickets, tiempos de resolución, desempeño de agentes y tendencias del servicio para apoyar decisiones basadas en datos en una mesa de ayuda.*

---

## Overview / Descripción general

This dashboard provides a clear view of IT support operations, helping teams monitor workload, identify bottlenecks, and measure service quality. It answers questions such as:

*Este dashboard ofrece una visión clara de las operaciones de soporte de IT, ayudando a los equipos a monitorear la carga de trabajo, identificar cuellos de botella y medir la calidad del servicio. Responde preguntas como:*

- How many tickets are we receiving over time? / ¿Cuántos tickets recibimos a lo largo del tiempo?
- What is the average resolution time? / ¿Cuál es el tiempo promedio de resolución?
- Which categories or issues are most common? / ¿Qué categorías o problemas son más frecuentes?
- How is each agent or team performing? / ¿Cómo es el desempeño de cada agente o equipo?

---

## Key Metrics & Features / Métricas y funciones clave

- **Total Tickets** — overall volume of support requests / volumen total de solicitudes de soporte
- **Average Resolution Time** — how long tickets take to close / tiempo promedio para cerrar tickets
- **Tickets by Category / Priority** — breakdown of issue types / desglose por tipo de problema
- **Tickets Over Time** — trend analysis by day, month, or quarter / análisis de tendencias por día, mes o trimestre
- **Agent / Team Performance** — workload and resolution comparison / comparación de carga y resolución
- **Interactive Filters** — slice by date, status, priority, and more / segmentación por fecha, estado, prioridad y más

## How to Use / Cómo usarlo

1. Download the `.pbix` file from this repository. / Descarga el archivo `.pbix` de este repositorio.
2. Open it with [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free). / Ábrelo con Power BI Desktop (gratuito).
3. If prompted, update the data source connection to your own data. / Si se solicita, actualiza la conexión al origen de datos.
4. Explore the report using the interactive filters and visuals. / Explora el reporte usando los filtros y visualizaciones interactivas.
